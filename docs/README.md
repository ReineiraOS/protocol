## Papers

| Document   | In this repo                                             | Canonical (hosted)                             |
| ---------- | -------------------------------------------------------- | ---------------------------------------------- |
| Whitepaper | [ReineiraOS-Whitepaper.pdf](./ReineiraOS-Whitepaper.pdf) | https://reineira.xyz/ReineiraOS-Whitepaper.pdf |

The whitepaper is the document referenced throughout the contract NatSpec (e.g.
`whitepaper §7.2`, `§7.10`). The in-repo copy is the snapshot that the code in
this commit was written against; the hosted copy at
[reineira.xyz](https://reineira.xyz) is the latest published version.
